Detailed Description

Modular and abstract class that spawns bullet in a pattern.

This function basically call the explode function on specified time. Every pattern of bullets will inherit of this class. A child of SpellCard will only have two functions to instantiate:

  • The constructor, which will have to call SpellCard() to specify how frequently new bullets will be added
  • Explode, this function will be called on the intervals specified by the constructor.

See the explode and spawn function to know what a child can do on those explosions.

Note also that update is final, as we want the child to only work during the explosions.
