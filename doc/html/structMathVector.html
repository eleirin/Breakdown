Detailed Description

An augmented class of sf::Vector.

This version allows us to have many more useful functions such as length, normalized, in time we will have dot products and many more

Note
{All functions are const, modification will only happen through direct copy affectation. This is to keep a mathematical scheme and expected behaviours.}
{All coordinates will be in the world coordinate and not in pixel, there will be one function to transcribe between the two}
